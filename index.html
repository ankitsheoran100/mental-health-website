A Multi-Task Deep Learning system for analyzing emotional sentiment and depression severity in text using DistilBERT and CORAL ordinal regression.

What This Tool Does

Two prediction tasks in one model — trained on combined mental health datasets

😊

Sentiment Analysis

Classifies text into Positive, Neutral, or Negative sentiment using the shared DistilBERT encoder with a 2-layer MLP head.

📊

Depression Severity

Predicts depression severity across 5 PHQ-9 levels — Minimum, Mild, Moderate, Moderately Severe, and Severe — using CORAL ordinal regression.

🔗

Cross-Task Attention

A cross-task attention gate allows sentiment predictions to guide severity detection, improving multi-task performance through shared representations.

Model Architecture

MultiTaskModel — DistilBERT backbone with dual task heads

🏗️

Shared Encoder

distilbert-base-uncased processes the input text and produces a 768-dimensional CLS token representation shared across both tasks.

⚡

Uncertainty-Weighted Loss

Kendall et al. (2018) uncertainty weighting automatically balances sentiment and severity losses during training using learnable log-sigma parameters.

📐

CORAL Ordinal Layer

Consistent Rank Logits (CORAL) ensures mathematically consistent ordinal predictions across the 5 severity classes without rank violations.
