Multi-Task DistilBERT for
Mental Health Analysis

AI-powered simultaneous detection of sentiment and depression severity using a single transformer model with cross-task attention gating and CORAL ordinal classification.

Key Features

🧠

Sentiment Classification

Classifies text into negative, neutral, or positive sentiment using a 3-layer MLP head (768→512→256→3) with LayerNorm and GELU.

📊

Depression Severity

Ordinal classification into minimum, mild, moderate, severe (PHQ-9 aligned) using CORAL layer with learnable per-boundary thresholds.

🔗

Cross-Task Gate

Sentiment logits pass through a 2-layer MLP sigmoid gate (3→128→256) to condition severity features, enabling multi-task information sharing.

⚖️

Uncertainty Weighting

Homoscedastic uncertainty loss (Kendall et al., CVPR 2018) with learnable log-sigma scalars (log_sigma_sentiment, log_sigma_severity) per task.

🎯

Learnable Thresholds

Per-boundary CORAL thresholds (coral_thresholds) are learned during training. 3 boundaries for 4 severity classes, clamped to (0.1, 0.9) at inference.

📈

Residual Connections

Two residual skip connections in the severity branch: after the cross-task gate and after severity_refine, preventing feature degradation.
