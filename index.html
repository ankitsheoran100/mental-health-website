Multi-Task DistilBERT for
Mental Health Analysis

Simultaneously detects sentiment and depression severity from text using a single transformer with cross-task attention gating and CORAL ordinal classification.

Key Features

🧠

Sentiment Classification

3-layer MLP (768→512→256→3) with LayerNorm + GELU predicts negative / neutral / positive.

📊

Depression Severity

CORAL ordinal head with learnable per-boundary thresholds predicts 4 PHQ-9 classes: minimum / mild / moderate / severe.

🔗

Cross-Task Gate

2-layer MLP gate (3→128→256+Sigmoid) fuses sentiment logits into severity features with a residual connection.

⚖️

Uncertainty Weighting

Homoscedastic uncertainty loss (Kendall 2018) with learnable log_sigma_sentiment and log_sigma_severity.

🎯

Learnable Thresholds

coral_thresholds — 3 per-boundary parameters learned during training, replacing fixed 0.5 cutoffs.

📈

Residual Connections

Two residual skip connections in the severity branch prevent feature degradation and improve gradient flow.

Output Labels

🧠 Sentiment (3-class)

😞 negative — class 0
😐 neutral — class 1
😊 positive — class 2

📊 Severity (4-class, PHQ-9)

🟢 minimum — class 0 (PHQ-9: 0–4)
🟡 mild — class 1 (PHQ-9: 5–9)
🟠 moderate — class 2 (PHQ-9: 10–14)
🔴 severe — class 3 (PHQ-9: 15–27)
