About This Project

A thesis research project combining Multi-Task Learning, Transformer models, and Ordinal Regression for mental health analysis.

Project Overview

This project presents a Multi-Task Deep Learning system that simultaneously predicts sentiment polarity and depression severity from text data. The system is built on a shared DistilBERT encoder with task-specific heads connected by a cross-task attention gate.

Model Architecture

Component Details
Encoder distilbert-base-uncased (66M parameters)
Hidden Size 768 dimensions (CLS token)
Sentiment Head 2-layer MLP (768 → 256 → 3 classes)
Severity Head CORAL ordinal layer (256 → 4 thresholds)
Cross-Task Gate Sigmoid gate projecting sentiment logits → severity features
Loss Function Uncertainty-weighted multi-task loss (Kendall et al., 2018)
Dropout 0.3 (shared encoder), 0.2 (severity refinement)
Max Sequence Length 128 tokens

Tasks

  • Sentiment Classification: Negative / Neutral / Positive (3 classes)
  • Depression Severity: Minimum / Mild / Moderate / Moderately Severe / Severe (5 PHQ-9 classes)

Datasets Used

  • Reddit mental health posts (sentiment labels)
  • DepressionEmo dataset (severity labels)
  • Combined and preprocessed for multi-task training

Preprocessing Pipeline

  • URL removal
  • Username anonymisation → [USER]
  • Subreddit tag removal → [COMMUNITY]
  • Repetition normalisation (e.g. sooooo → so)
  • Contraction expansion (can't → cannot)
  • Whitespace normalisation

Baseline Models Compared

  • Logistic Regression (TF-IDF features)
  • Linear SVC (TF-IDF features)
  • Single-Task DistilBERT (sentiment only)
  • Single-Task CORAL DistilBERT (severity only)
  • MultiTaskModel (proposed — best performance)

References

  • Kendall, A., Gal, Y., & Cipolla, R. (2018). Multi-task learning using uncertainty to weigh losses.
  • Cao, W. et al. (2020). Rank consistent ordinal regression for neural networks (CORAL).
  • Sanh, V. et al. (2019). DistilBERT, a distilled version of BERT.
