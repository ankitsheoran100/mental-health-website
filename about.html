About This Project

Multi-Task DistilBERT for Joint Sentiment and Depression Severity Detection

🎓 Thesis Overview

This project presents a multi-task learning framework for simultaneously detecting sentiment (negative / neutral / positive) and depression severity (minimum / mild / moderate / severe) from text using a single fine-tuned DistilBERT model.

The key contribution is a cross-task attention gate that routes sentiment information into the severity branch, enabling the model to leverage sentiment context when predicting depression severity. Learnable per-boundary CORAL thresholds replace the fixed 0.5 threshold used in standard CORAL.

🏷️ Label Definitions

Sentiment (3-class)

😞 negative — class 0
😐 neutral — class 1
😊 positive — class 2

Severity (4-class, PHQ-9 Aligned)

🟢 minimum — class 0 (PHQ-9: 0–4)
🟡 mild — class 1 (PHQ-9: 5–9)
🟠 moderate — class 2 (PHQ-9: 10–14)
🔴 severe — class 3 (PHQ-9: 15–27)
