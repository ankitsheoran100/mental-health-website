How It Works

1

Text Input

Text is tokenised with DistilBERT's WordPiece tokeniser. MAX_LEN=256 tokens, padding/truncation applied.

2

Encoding

DistilBERT encoder extracts [CLS] token representation (768-dim) from all 256 tokens.

3

Sentiment Head

3-layer MLP (768→512→256→3) with LayerNorm + GELU predicts sentiment class (0–2).

4

Cross-Task Gate

Sentiment logits pass through 2-layer MLP gate (3→128→256) to condition severity features with residual connection.

5

CORAL + Thresholds

Gated severity features → severity_refine → CORAL ordinal output. Learnable per-boundary thresholds (not fixed 0.5).

6

Results

Returns sentiment class (0–2) and severity class (0–3) simultaneously with confidence scores.
